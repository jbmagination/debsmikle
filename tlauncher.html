Downloading the modpack

You need Minecraft: Java Edition and a compatible launcher (see section below) in order to use this modpack.

You can download the modpack by clicking here.

"Installing" the modpack for TLauncher users

First, press the TLMods button. Choose the Create button at the top, and write "debsmikle" as the name. Set the game version to 1.19.1 if it's not set to that already - everything else can be as-is. Choose Create.

"debsmikle" should now be a selected modpack. Open the dropdown, and select the management icon next to "debsmikle". Then, choose Open mod-pack folder.

Extract the .zip and open the contents of the .minecraft folder. Copy or move the contents of that folder into the folder that was just opened in TLauncher.

Under which version you want to select for launching Minecraft, choose Release debsmikle. You can now launch the game as normal.

Note: TLauncher skins and capes are not supported, as they (usually) pulled off with a mod for the Forge mod loader. This modpack uses the Fabric mod loader.
