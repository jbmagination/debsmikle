Downloading the modpack

You need Minecraft: Java Edition and a compatible launcher (see section below) in order to use this modpack.

You can download the modpack by clicking here.

Compatible launchers

This modpack is compatible with any of the following launchers:

  • PolyMC
  • MultiMC
  • ManyMC
  • GDLauncher
  • CurseForge
  • ATLauncher
  • ...or any other launcher that claims to support any of the above

"Installing" the modpack

PolyMC, MultiMC, and ManyMC users: Press Add Instance, then on the sidebar choose Import from zip. Select the .zip file in your files, then choose OK. You can then open the instance as normal.

GDLauncher users: Press the + in the bottom-left corner to add a new instance, then select Import Zip from the top bar. Select the .zip file in your files, then choose OK and go to the next page. Set a name (or go with the default "debsmikle"), then go to the next page. Once everything finishes downloading, you can open the instance as normal.

CurseForge users: Under the game Minecraft, choose Create Custom Profile. Then, choose Or import a previously created profile. Once everything finishes downloading, you can open the profile as normal.

ATLauncher users: Under the Instances tab, press the Import button in the top-left corner. Select the .zip file in your files, then choose Import. Leave everything as-is and choose Install. Once the instance finishes being imported, you can open the instance as normal.
